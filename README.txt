# GoMod_BlenderAddon
Blender Python Addon, creates and simplifies the creation of modular assets and helps managing them.

Go Modular is a fast way for creating modular assets and previsualizing, for sculpting and editing the mesh. Also there´s usefull tools like converting to peaces and converting into a single mesh, also group managment and arranging modular version into a line or grid. Give it a try, it´s still under developement so future updates are on the way, feed back is welcome.
